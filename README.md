# HexaEdit

Simple hex editor made in C++ with Dear ImGui, SDL3 and OpenGL.

This project is made for learning:

* C++
* GUI programming
* binary file work
* rendering
* memory handling

Current version can:

* open files
* show file data in hexadecimal view
* show ASCII text preview
* scroll large files
* use file dialog

## Build

### Requirements

* CMake
* C++ compiler
* SDL3
* OpenGL

### Build project

```bash
mkdir build
cd build

cmake ..
cmake --build .
```
